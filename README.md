# aws-cdk-find-space

## CDK command
###  Initialize
```ckd init app --language typescript```
###  Synth with cloudformation
```cdk synth```
###  Initial cloud bootstrap
```cdk bootstrap```
### deploy
```
cdk deploy
cdk deploy --parameters duration=12
```

### Delete stack
``` cdk destroy```

### deploy all stacks
``` cdk deploy --all ```

### list all stacks
``` cdk list```

### check change set
```cdk diff```

### check health
```cdk doctor```
